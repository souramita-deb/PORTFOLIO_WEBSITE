## My Portfolio Website

This repository contains my personal portfolio website, developed as part of my journey as a frontend developer. The project reflects my current skills, creativity, and understanding of modern web development fundamentals.

It was built independently from scratch without using any pre-designed templates.

----

## Overview

The portfolio is designed with a custom layout rather than a conventional structure. It includes all essential portfolio sections while maintaining a unique visual approach.

The project focuses on:

- Theme customization (Light/Dark mode)

- Clean UI structure

- Responsive design

- Basic animations for improved user interaction

----

## Features

- Light/Dark theme toggle using JavaScript

- Theme control implemented with CSS variables

- Responsive design using media queries

- Smooth CSS transitions and animations

- DOM manipulation with event listeners

- Structured separation of HTML, CSS, and JavaScript

- Semantic HTML layout

----

## Technologies Used

- HTML5

- CSS3

- JavaScript
